can run programs using given bash script (on Linux)
  e.g. program triangle.c
  cd Language
  ./../run.sh triangle


Language
  motion.c - moving white rectangle
  triangle.c - produce simple, multi-colored triangle

Redbook
  contains sample files from the OpenGL red book

System
  sampleGUI.c - 3D OpenGL with GUI interface for adjustment


* all programs use C, OpenGL, GLUT, etc. *

Multi-platform support:

WINDOWS (cygwin)
  gcc -o FILE.exe FILE.c -lgl -lglu -lglut -lopenl32
  startx
    gives you xterm window
  ./FILE.exe


MAC
  gcc -o FILE FILE.c -framework GLUT -framework OpenGL
  ./FILE
